## .net files for Jenkins CI/CD Course

Install .net SDK in Ubuntu22:
```
sudo apt install dotnet-sdk-6.0
dotnet --version
```
create a sample app:
```
dotnet new mvc --name dotnetwebapp
```
Run the application:
```
cd dotnetwebapp
dotnet run /or dotnet run --urls http://*:5000
```
